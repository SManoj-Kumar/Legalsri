# Legalsri – Legal Services Website

A production-ready Next.js 14 website for Legalsri legal services platform.

## 🚀 Local Setup

```bash
npm install
npm run dev
# Open http://localhost:3000
```

## 📁 All Pages

| Route             | Page                 |
| ----------------- | -------------------- |
| `/`               | Homepage             |
| `/services?tab=0` | Company Registration |
| `/services?tab=1` | GST Services         |
| `/services?tab=2` | Trademark & IP       |
| `/services?tab=3` | ROC Compliance       |
| `/services?tab=4` | Startup Advisory     |
| `/services?tab=5` | Legal Drafting       |
| `/how-it-works`   | Process Page         |
| `/blog`           | Blog Hub             |
| `/privacy`        | Privacy Policy       |
| `/terms`          | Terms & Conditions   |
| `/cookies`        | Cookie Policy        |
| `/*`              | 404 Not Found        |
